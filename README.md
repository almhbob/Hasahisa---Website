# Hasahisa---Website
موقع خاص في مدينة الحسحيسة

هذا المشروع هو موقع ويب احترافي يعرض موقع الحسحيسة الخاص في مدينة الحسحيسة. الموقع مبني باستخدام HTML، CSS، وJavaScript ليوفر تجربة مستخدم سلسة وجذابة.

## الميزات
- تصميم احترافي مع دعم اللغة العربية (RTL)
- تنقل سلس بين الأقسام
- أقسام: عن الموقع، المعرض، اتصل بنا
- تصميم متجاوب

## كيفية التشغيل
افتح ملف `index.html` في متصفح الويب، أو شغل خادم محلي:

```bash
python3 -m http.server 8000
```

ثم اذهب إلى http://localhost:8000
